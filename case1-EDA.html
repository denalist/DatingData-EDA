Lets engineer a variable to simplify pet ownership status.

#Pets: split into three categories, have, don't have but want, don't want
like_pets = unique(profiles$pets)[c(1,3,7,10)]
dislike_pets = unique(profiles$pets)[c(13,14,15,16)]
has_pets = unique(profiles$pets)[c(2,5,6,8,9,11,12)]
for (i in 1:nrow(profiles)){
if(profiles$pets[i] %in% has_pets){
  profiles$pet_factor[i] = 'Has Pets'
}
if(profiles$pets[i] %in% dislike_pets){
  profiles$pet_factor[i] = 'Dislikes Pets'
}
if(profiles$pets[i] %in% like_pets){
  profiles$pet_factor[i] = 'Likes Pets'
}
}

Now lets do the same for simplifying a users offspring status.

Insight #1: LGBT community

We are interested in finding the areas with the highest populations of LGBT users.

Here lets calculate the number of LGBT users in each county.

lgbt = profiles[(profiles$orientation=='bisexual')|(profiles$orientation=='gay'),]
counties$lgbt_users = 0
for(i in 1:nrow(counties)){
  x = lgbt[lgbt$county==counties$county[i],]
  counties$lgbt_users[i] = nrow(x)
}
counties$lgbt_proportion = counties$lgbt_users / counties$num_users
